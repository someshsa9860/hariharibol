PHASE 1: Preparation (1-2 hours)
  □ Create AWS account
  □ Verify free tier eligibility
  □ Gather API keys (Gemini, Firebase, OpenAI)
  □ Prepare domain registration
  □ Generate JWT secrets
  □ Create .env template

PHASE 2: EC2 Setup (30 minutes)
  □ Launch t3.micro instance (Ubuntu 24.04)
  □ Download .pem key
  □ SSH access confirmed
  □ Security groups configured

PHASE 3: Install Services (45 minutes)
  □ Node.js v20
  □ PostgreSQL 16
  □ Nginx
  □ PM2 (process manager)
  □ Certbot (SSL)

PHASE 4: Deploy Apps (1-2 hours)
  □ Clone repository
  □ Backend: install → build → migrate → start
  □ Admin Panel: install → build → start
  □ Website: install → build → start

PHASE 5: Configure Nginx (30 minutes)
  □ Reverse proxy setup
  □ 3 upstream services configured
  □ Domain routing working
  □ HTTP → HTTPS redirect

PHASE 6: SSL Certificates (15 minutes)
  □ Let's Encrypt setup
  □ Auto-renewal enabled
  □ All domains have HTTPS

PHASE 7: Storage & Backups (20 minutes)
  □ Local storage directory created
  □ Backup script written
  □ Daily cron job configured
  □ 7-day retention policy

PHASE 8: Testing & Launch (30 minutes)
  □ Website loads over HTTPS
  □ API responds
  □ Admin panel accessible
  □ Database connected
  □ All 3 apps running (pm2 status)
